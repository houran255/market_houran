### 项目：Java超市管理系统
#### 作者：侯冉
此项目为超市管理系统，由笔者独自于2023年3月至4月完成，耗时3周，使用Java和MySQL语言，现将它整理公开。

直接打包项目到本地，然后编译运行Login_sql.java文件
